# Sales-Revenue-Performance-Analysis
Developed an interactive sales performance dashboard in Data Studio to monitor key business metrics, revenue trends, and customer behavior. 
The dashboard provides KPI tracking, revenue analysis by sales channel, city, and loyalty segment, highlights top-performing products, and supports dynamic filtering by date, payment method, payment status, sales channel, and loyalty status. 
It enables stakeholders to quickly identify performance trends and make data-driven business decisions.
